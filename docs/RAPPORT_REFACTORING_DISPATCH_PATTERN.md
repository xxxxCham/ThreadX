## üîß Solution Impl√©ment√©e

### Nouvelle M√©thode `_dispatch_indicator()`

**Localisation** : Ligne 153 (apr√®s `_should_use_gpu_dynamic()`)

**Signature** :
```python
def _dispatch_indicator(
    self,
    indicator_name: str,
    data: pd.DataFrame,
    params: Dict[str, Any],
    use_gpu: Optional[bool],
    gpu_func: Callable,
    cpu_func: Callable,
    input_cols: Optional[str] = None,
    extract_arrays: bool = True,
) -> Any:
    """
    Dispatch automatique GPU/CPU pour un indicateur.

    Centralise la logique de d√©cision GPU/CPU et l'extraction de donn√©es
    pour √©viter la duplication de code entre indicateurs.
    """
```

**Workflow** :
1. **D√©tection dtype** : Extrait le dtype de la colonne sp√©cifi√©e
2. **D√©cision GPU/CPU** : Appelle `_should_use_gpu_dynamic()` si `use_gpu=None`
3. **Extraction donn√©es** : Convertit en `np.ndarray` si `extract_arrays=True`
4. **Dispatch** : Appelle `gpu_func` ou `cpu_func` selon la d√©cision
5. **Retour r√©sultat** : Retourne directement le r√©sultat (Series ou Tuple)

**Flexibilit√©** :
- ‚úÖ Support colonnes simples (`input_cols='close'`)
- ‚úÖ Support DataFrame complet (`extract_arrays=False`)
- ‚úÖ Support fonctions avec closures (lambdas pour passer `period`, `index`, etc.)

## üîç D√©tails Techniques

### Lambda Closures vs M√©thodes Partielles

**Choix fait** : Lambdas
```python
gpu_func=lambda prices: self._rsi_gpu(prices, period, data.index)
```

**Alternative** : `functools.partial`
```python
from functools import partial
gpu_func=partial(self._rsi_gpu, period=period, index=data.index)
```

**Pourquoi Lambdas ?**
1. ‚úÖ Plus lisible (pas d'import suppl√©mentaire)
2. ‚úÖ Flexibilit√© ordre des arguments
3. ‚úÖ Debugging plus facile (stacktrace montre lambda)
4. ‚úÖ M√™me performance (compiled une fois)

### Gestion des Types

**Type Hints** :
```python
gpu_func: Callable  # Accepte toute fonction
cpu_func: Callable  # Accepte toute fonction
```

**Flexibilit√©** : Accepte :
- Fonctions simples : `self._rsi_gpu`
- Lambdas : `lambda x: self._rsi_gpu(x, period)`
- Partials : `partial(self._rsi_gpu, period=14)`
- M√©thodes : `self._method`

### Extraction de Donn√©es

**Deux modes** :

1. **Extraction colonne** (`extract_arrays=True`) :
```python
input_cols='close'
‚Üí arrays = np.asarray(data['close'].values)
```

2. **DataFrame complet** (`extract_arrays=False`) :
```python
input_cols=None, extract_arrays=False
‚Üí arrays = data  # Passe le DataFrame
```
